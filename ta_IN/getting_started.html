Qiskit மூலம் முதலில் தொடங்குதல்¶

Qiskitடைப் பயன்படுத்துவதற்கான வேலை ஏற்பாடு(workflow) மூன்று உயர் மட்ட படிகளைக் கொண்டுள்ளது:

  • ** உருவாக்கு **: நீங்கள் கருத்தில் கொண்ட சிக்கலைக் குறிக்கும் குவாண்டம் சுற்று வடிவமைக்கவும்.

  • ** இயக்கவும் **: வெவ்வேறு பின்தளத்தில் சோதனைகளை இயக்கவும் (* இதில் கணினி இயந்திரம் மற்றும் சிமுலேட்டர்கள் இரண்டும் அடங்கும் *).

  • ** பகுப்பாய்வு **: சுருக்கமான புள்ளிவிவரங்களைக் கணக்கிட்டு, சோதனைகளின் முடிவுகளைக் காட்சிப்படுத்துங்கள்.

முழு செயல்முறைக்கும் ஒரு எடுத்துக்காட்டு இங்கே கொடுக்கப்பட்டு உள்ளது, ஒவ்வொரு பகுதியும் அடுத்தடுத்த பிரிவுகளில் விரிவாக விளக்கப்பட்டுள்ளது:

import numpy as np
from qiskit import(
  QuantumCircuit,
  execute,
  Aer)
from qiskit.visualization import plot_histogram

# Use Aer's qasm_simulator
simulator = Aer.get_backend('qasm_simulator')

# Create a Quantum Circuit acting on the q register
circuit = QuantumCircuit(2, 2)

# Add a H gate on qubit 0
circuit.h(0)

# Add a CX (CNOT) gate on control qubit 0 and target qubit 1
circuit.cx(0, 1)

# Map the quantum measurement to the classical bits
circuit.measure([0,1], [0,1])

# Execute the circuit on the qasm simulator
job = execute(circuit, simulator, shots=1000)

# Grab results from the job
result = job.result()

# Returns counts
counts = result.get_counts(circuit)
print("\nTotal count for 00 and 11 are:",counts)

# Draw the circuit
circuit.draw()


Total count for 00 and 11 are: {'00': 490, '11': 510}

q_1: ─────┤ X ├─╫─┤M├

c: 2/═══════════╩══╩═
                0  1 
# Plot a histogram
plot_histogram(counts)

செயல்முறை படி படியாக விளக்கப்பட்டுள்ளது¶

மேலே நிரல் கீழே ஆறு படிநிலைகளாக உடைந்து கொள்ளலாம்:

  1. கோப்பு தொகுப்புகள்

  2. மாறிகளை துவக்குக

  3. கேட்களைச் சேர்க்கவும்

  4. சுற்றுப்பாதை காட்சிப்படுத்தவும்

  5. பரிசோதனையை உருவகப்படுத்துங்கள்

  6. முடிவுகளைக் காண்பிக்கவும்

Step 1: தொகுப்புகளை இறக்குமதி செய்க¶

உங்கள் Program க்கு தேவையான அடிப்படை தனிமங்கள் பின்வருமாறு இறக்குமதி செய்யப்படுகின்றன:

import numpy as np
from qiskit import(
  QuantumCircuit,
  execute,
  Aer)
from qiskit.visualization import plot_histogram

இன்னும் விரிவாக, இறக்குமதிகள்

  • QuantumCircuit: குவாண்ட் அமைப்பின் ஆணைகள் என நினைத்துக் கொள்ளலாம். இது அனைத்து உங்கள் குவாண்ட் செயல்பாடுகள் உள்ளது.

  • execute: உங்கள் சுற்றுப்பாதை/பரிசோதனையை இயக்குகிறது.

  • Aer: சிமுலேட்டர் பின்தளத்தில் கையாளுகிறது.

  • ` ` plot_ histogram ` `: ஹிஸ்டோகிராம் உருவாக்குகிறது.

Step 2: மாறிகளை துவக்குக¶

குறியீட்டின் அடுத்த கோட்டை குறிக்கவும்

circuit = QuantumCircuit(2, 2)

இங்கே, நீங்கள் பூஜ்ஜிய நிலையில் 2 குவிட்டுகளுடன் துவக்குகிறீர்கள்; 2 கிளாசிக்கல் பிட்கள் பூஜ்ஜியமாக அமைக்கப்பட்டன; மற்றும் `` சுற்று`` என்பது குவாண்டம் சுற்று.

வாக்கிய அமைப்பு:

  • ` ` QuantumCircuit(int, int) ` `

Step 3 : கேட்ஸ் சேர்க்கவும்¶

உங்கள் சுற்றுப்பின் பதிப்பை கையாளும் கேட்கள் (செயல்பாடுகள்) சேர்க்க முடியும்.

பின்வரும் மூன்று வரிகளை கருத்தில் கொள்ளுங்கள்:

circuit.h(0)
circuit.cx(0, 1)
circuit.measure([0,1], [0,1])

கேட்கள் ஒன்று சேர்க்கப்பட்டு ஒன்று-ஒன்று மூலம் பெல் ஸ்டேடை வடிவமைக்கப் படுகிறது

\[|\psi\rangle = \left(|00\rangle+|11\rangle\right)/\sqrt{2}.\]

மேலே குறியீடு பின்வரும் கேட்களை பயன்படுத்துகிறது:

  • QuantumCircuit.h(0): A Hadamard gate \(H\) இது குபிட் 0 ஐ superposition state வைக்கிறது.

  • `` QuantumCircuit.cx (0, 1) ``: கட்டுப்பாட்டு குவிட் 0 மற்றும் இலக்கு குவிட் 1 இல் கட்டுப்படுத்தப்பட்ட-செயல்படாத செயல்பாடு (:math: CNOT), குவிட்களை ** entangled state** நிலையில் வைக்கிறது.

  • `` QuantumCircuit.measure ([0,1], [0,1]) ``: நீங்கள் முழு quantum மற்றும் கிளாசிக்கல் பதிவேடுகளை ``measure'' க்கு அனுப்பினால், ith qubit இன் அளவீட்டு முடிவு ith கிளாசிக்கல் பிட்டில் சேமிக்கப்படும்.

Step 4: மின்சுற்று காண்க¶

நீங்கள் பயன்படுத்தலாம் qiskit.circuit.QuantumCircuit.draw() பல பாடப்புத்தகங்கள் மற்றும் ஆராய்ச்சி கட்டுரைகளில் பயன்படுத்தப்படும் பல்வேறு வடிவங்களில் நீங்கள் வடிவமைத்துள்ள சுற்றுவட்டத்தைக் காண.

circuit.draw()

     ┌───┐     ┌─┐   
q_0: ┤ H ├──■──┤M├───
     └───┘┌─┴─┐└╥┘┌─┐
q_1: ─────┤ X ├─╫─┤M├
          └───┘ ║ └╥┘
c: 2/═══════════╩══╩═
                0  1

இந்த சுற்றில், குவிட்கள் மேலே குவிட் பூஜ்ஜியத்துடன் கட்டளையிடப்படுகின்றன, மேலும் கீழே ஒரு குவிட். சுற்று இடமிருந்து வலமாக படிக்கப்படுகிறது, அதாவது சுற்றுக்கு முன்னர் பயன்படுத்தப்பட்ட வாயில்கள் இடதுபுறமாகக் காண்பிக்கப்படுகின்றன.

`` QuantumCircuit.draw () `` அல்லது `` qiskit.visualization.circuit_drawer () `` க்கான இயல்புநிலை பின்தளத்தில் உரை பின்தளத்தில் உள்ளது. இருப்பினும், உங்கள் உள்ளூர் சூழலைப் பொறுத்து, இந்த இயல்புநிலைகளை உங்கள் பயன்பாட்டு வழக்குக்கு மிகவும் பொருத்தமானதாக மாற்ற விரும்பலாம். இது பயனர் கட்டமைப்பு கோப்பில் செய்யப்படுகிறது. முன்னிருப்பாக பயனர் கட்டமைப்பு கோப்பு `` ~ /.qiskit / settings.conf`` இல் இருக்க வேண்டும், இது .ini கோப்பாகும்.

உதாரணமாக, settings.conf மேட்லோட்லிப் வரைபடம் அமைப்பதற்கான கோப்பு:

[default]
circuit_drawer = mpl

இந்த வடிவமைப்புக்கான மதிப்பு போன்றவற்றில் நீங்கள் செல்லுபடியான வரையறுக்கப்பட்ட வரைபடத்தை பயன்படுத்தலாம், இது உரை, mpl, latex, மற்றும் latex_source ஆகியவற்றை உள்ளடக்குகிறது.

Step 5: பரிசோதனையை உருவகப்படுத்துங்கள்¶

Qiskit Aer என்பது Quantum சுற்றுகளுக்கான உயர் செயல்திறன் சிமுலேட்டர் கட்டமைப்பாகும். several backends <apidoc/providers/aer/aer_backends.html> வெவ்வேறு உருவகப்படுத்துதல் இலக்குகளை அடைய இது வழங்குகிறது.

Aer ஐ நிறுவுவதில் சிக்கல் இருந்தால், Aer ஐ` BasicAer` உடன் மாற்றுவதன் மூலம் நீங்கள் அடிப்படை Aer வழங்குநரைப் பயன்படுத்தலாம். Qiskit டெர்ராவில் அடிப்படை Aer சேர்க்கப்பட்டுள்ளது.

import numpy as np
from qiskit import(
  QuantumCircuit,
  execute,
  BasicAer)
...

இந்த சுற்றுப்பாதை உபயோகிக்கவும், நீங்கள் ` ` qasm_ simulator ` ` `. ஒவ்வொரு இயங்கும் இந்த சுற்றுப்பாதை பயன்படுத்தும் பிட் சரம் 00 அல்லது 11.

simulator = Aer.get_backend('qasm_simulator')
job = execute(circuit, simulator, shots=1000)
result = job.result()
counts = result.get_counts(circuit)
print("\nTotal count for 00 and 11 are:",counts)


Total count for 00 and 11 are: {'00': 520, '11': 480}

எதிர்பார்த்தபடி, வெளியீட்டு பிட் சரம் 00 சுமார் 50 சதவீதம் நேரம். சர்க்யூட் இயங்கும் எத்தனை முறை `` execute`` முறையின் shots வாதத்தின் மூலம் குறிப்பிடப்படலாம். உருவகப்படுத்துதலின் காட்சிகளின் எண்ணிக்கை 1000 ஆக அமைக்கப்பட்டது (இயல்புநிலை 1024).

உங்களிடம் `` result`` பொருள் கிடைத்ததும், `` get_counts (circuit) `` முறை மூலம் எண்ணிக்கையை அணுகலாம். இது நீங்கள் ஓடிய சோதனையின் மொத்த விளைவுகளை வழங்குகிறது.

Step 4: முடிவுகளைக் காட்சிப்படுத்து¶

Qiskit ` many visualizations <apidoc/visualization.html>` வழங்குகிறது _,

செயல்பாடு ` plot_histogram ` `, உங்கள் முடிவுகளைக் காண.

plot_histogram(counts)

கவனிக்கப்பட்ட நிகழ்தகவுகள் \(Pr(00)\) மற்றும் \(Pr (11)\) அந்தந்த எண்ணிக்கையை எடுத்து மொத்த காட்சிகளின் எண்ணிக்கையால் வகுப்பதன் மூலம் கணக்கிடப்படுகின்றன.

Note

மதிப்பிடப்பட்ட நிகழ்தகவுகள் எவ்வாறு மாறுகின்றன என்பதைக் காண `` execute`` முறையில் `` shots`` முக்கிய சொல்லை மாற்ற முயற்சிக்கவும்.
