*Optimisasi Portofolio *¶

Pengantar¶

Tutorial ini menunjukkan bagaimana caranya untuk menyelesaikan masalah optimisasi mean-varians portofolio untuk :math:` n ` aset:

:math:` begin{aligned} min_ {x in {0, 1 } ^ n} q x ^ T Sigma x-mu ^ T x text{subject to: } 1 ^ T x = B end{aligned}`

di mana kita menggunakan notasi sebagai berikut:

  • :math:` x in {0, 1 } ^ n ` menandakan vektor dari variabel keputusan biner, yang mengindikasikan aset mana yang akan diambil (:math:` x[i] = 1 ) dan mana yang tidak diambil (:math: x[i] = 0 `),

  • :math:` mu in mathbb{R}^ n ` mendefinisikan hasil investasi atau return dari aset-aset,

  • :math:` Sigma in mathbb{R}^ {n times n} ` merupakan kovariansi antara aset,

  • :math:` q > 0 ` mengendalikan derajat risiko atau risk appetite dari pembuat keputusan,

  • dan :math:` B ` menandakan anggaran atau budget, yaitu jumlah aset yang akan dipilih dari :math:` n `.

Kita asumsikan penyederhanaan berikut:-semua aset memiliki harga yang sama (dinormalisasikan menjadi 1),–anggaran penuh :math:` B ` harus dikeluarkan semua, yaitu seseorang harus memilih tepat :math:` B ` aset.

Batasan atau constraint persamaan :math:` 1 ^ T x = B ` dipetakan ke dalam sebuah istilah penalti :math:` (1 ^ T x-B) ^ 2 ` dengan suatu parameter sebagai skala dan dikurangkan dari fungsi objektif. Masalah yang dihasilkan dapat dipetakan ke Hamiltonian dengan keadaan dasar yang sesuai dengan solusi optimal. Notebook ini menunjukkan bagaimana caranya menggunakan Variational Quantum Eigensolver (VQE) atau Quantum Approximate Optimization Algorithm (QAOA) untuk menemukan solusi optimal pada satu set parameter yang diberikan.

Percobaan pada perangkat keras kuantum nyata untuk masalah ini dilaporkan sebagai contoh dalam artikel berikut: Improving Variational Quantum Optimization using CVaR. Barkoutsos et al. Tahun 2019.

from qiskit import Aer
from qiskit.circuit.library import TwoLocal
from qiskit.aqua import QuantumInstance
from qiskit.finance.applications.ising import portfolio
from qiskit.optimization.applications.ising.common import sample_most_likely
from qiskit.finance.data_providers import RandomDataProvider
from qiskit.aqua.algorithms import VQE, QAOA, NumPyMinimumEigensolver
from qiskit.aqua.components.optimizers import COBYLA
import numpy as np
import matplotlib.pyplot as plt
import datetime

Menentukan contoh masalah¶

Di sini sebuah instansi Operator dibuat untuk Hamiltonian kita. Dalam kasus ini, matriks Pauli didapatkan dari sebuah Hamiltonian Ising yang diterjemahkan dari masalah portofolio. Kami menggunakan masalah portofolio acak untuk notebook ini. Perluasan ke masalah data finansial nyata merupakan hal yang mudah seperti yang diilustrasikan di sini: ` Loading dan Processing Stock-Market Time-Series Data <11_time_series.ipynb> ` __

[2]:

# set number of assets (= number of qubits)
num_assets = 4

# Generate expected return and covariance matrix from (random) time-series
stocks = [("TICKER%s" % i) for i in range(num_assets)]
data = RandomDataProvider(tickers=stocks,
                 start=datetime.datetime(2016,1,1),
                 end=datetime.datetime(2016,1,30))
data.run()
mu = data.get_period_return_mean_vector()
sigma = data.get_period_return_covariance_matrix()

Kita mendefinisikan beberapa metode utilitas untuk mencetak hasilnya dalam format yang baik.

[5]:

def index_to_selection(i, num_assets):
    s = "{0:b}".format(i).rjust(num_assets)
    x = np.array([1 if s[i]=='1' else 0 for i in reversed(range(num_assets))])
    return x

def print_result(result):
    selection = sample_most_likely(result.eigenstate)
    value = portfolio.portfolio_value(selection, mu, sigma, q, budget, penalty)
    print('Optimal: selection {}, value {:.4f}'.format(selection, value))

    eigenvector = result.eigenstate if isinstance(result.eigenstate, np.ndarray) else result.eigenstate.to_matrix()
    probabilities = np.abs(eigenvector)**2
    i_sorted = reversed(np.argsort(probabilities))
    print('\n----------------- Full result ---------------------')
    print('selection\tvalue\t\tprobability')
    print('---------------------------------------------------')
    for i in i_sorted:
        x = index_to_selection(i, num_assets)
        value = portfolio.portfolio_value(x, mu, sigma, q, budget, penalty)
        probability = probabilities[i]
        print('%10s\t%.4f\t\t%.4f' %(x, value, probability))

NumPyMinimumEigensolver (sebagai referensi klasik)¶

Mari pecahkan masalahnya. Pertama secara klasik…

Kita sekarang dapat menggunakan Operator yang kita buat di atas tanpa memperhatikan secara spesifik bagaimana hal itu diciptakan. Kita menetapkan algoritma untuk NumPyMinimumEigensolver sehingga kita dapat memiliki referensi klasik. Masalah ini diatur untuk 'ising'. Backend tidak diperlukan karena persoalan ini dihitung secara klasik dan tidak menggunakan komputasi kuantum. Hasilnya dikembalikan dalam bentuk kamus atau dictionary.

[6]:

exact_eigensolver = NumPyMinimumEigensolver(qubitOp)
result = exact_eigensolver.run()

print_result(result)

Solusi menggunakan VQE¶

Sekarang kita bisa menggunakan Variational Quantum Eigensolver (VQE) untuk memecahkan masalah tersebut. Kita akan menetapkan bentuk optimasi dan bentuk variasional yang akan digunakan.

Catatan: Anda dapat mengubah ke backends lainnya dengan memberikan nama backend.

[7]:

backend = Aer.get_backend('statevector_simulator')
seed = 50

cobyla = COBYLA()
cobyla.set_options(maxiter=500)
ry = TwoLocal(qubitOp.num_qubits, 'ry', 'cz', reps=3, entanglement='full')
vqe = VQE(qubitOp, ry, cobyla)
vqe.random_seed = seed

quantum_instance = QuantumInstance(backend=backend, seed_simulator=seed, seed_transpiler=seed)

result = vqe.run(quantum_instance)

print_result(result)

Solusi menggunakan QAOA¶

Kita juga menunjukkan hasil penggunaan Quantum Approximate Optimization Algorithm (QAOA) di sini. Algoritma ini merupakan sebuah algoritma variasional lainnya dan menggunakan bentuk variasional internal yang dibuat berdasarkan masalah tersebut.

[8]:

backend = Aer.get_backend('statevector_simulator')
seed = 50

cobyla = COBYLA()
cobyla.set_options(maxiter=250)
qaoa = QAOA(qubitOp, cobyla, 3)

qaoa.random_seed = seed

quantum_instance = QuantumInstance(backend=backend, seed_simulator=seed, seed_transpiler=seed)

result = qaoa.run(quantum_instance)

print_result(result)
