Cài đặt Qiskit¶

Yêu cầu¶

Qiskit hỗ trợ Python từ phiên bản 3.6. Tuy nhiên, cả Python và Qiskit luôn được cập nhật, và đôi khi giữa các phiên bản sẽ có những điểm không tương thích.

If you're having trouble installing or using Qiskit after updating Python, check the Qiskit Package metadata for Programming Language to see which specific versions of Python are supported.

Chúng tôi khuyên bạn nên cài đặt Anaconda, một phiên bản Python đa nền tảng cho các tính toán trong nghiên cứu khoa học. Anaconda đã tích hợp sẵn Jupyter (Notebook), được khuyên nên dùng để tương tác với Qiskit.

Qiskit đã được thử nghiệm và hỗ trợ trên các hệ thống 64-bit sau:

  • Ubuntu 16.04 trở lên

  • macOS 10.12.6 hoặc mới hơn

  • Windows 7 hoặc mới hơn

Cài đặt¶

Chúng tôi đề nghị sử dụng môi trường Python ảo để tách Qiskit khỏi các ứng dụng khác và cải thiện trải nghiệm của bạn.

Cách đơn giản nhất để sử dụng các môi trường là dụng lệnh conda, nếu cài đặt Anaconda. Một môi trường Conda cho phép bạn sử dụng một phiên bản Python và các thư viện mong muốn. Mở một terminal (màn hình chạy lệnh) trong thư mục mà bạn muốn làm việc.

Các bạn nên sử dụng màn hình lệnh Anaconda Prompt được cài đặt với Anaconda. Tất cả những gì bạn phải làm là tạo một môi trường ảo bên trong Anaconda và kích hoạt nó. Các lệnh này có thể chạy trong Anaconda Prompt bất kể máy Windows hoặc Linux.

Tạo môi trường tối giản với chỉ có Python được cài đặt.

conda create -n name_of_my_env python=3

Kích hoạt môi trường mới của bạn.

conda activate name_of_my_env

Ghi chú

Nếu bạn đang sử dụng các phiên bản conda trước 4.6, sử dụng: Đối với Linux hoặc macOS: source activate name_of_my_env Đối với Windows: activate name_of_my_env

Tiếp theo, cài đặt Qiskit, bao gồm Terra, Aer, Ignis, và Aqua.

pip install qiskit

Ghi chú

Từ phiên bản Qiskit 0.13.0, cần có pip 19 hoặc mới hơn để cài đặt qiskit-aer từ file binary trên Linux. Nếu bạn chưa cài đặt pip 19, bạn có thể chạy command pip install -U pip để cài đặt nó. Nếu không có pip 19 hoặc với bản pip mới hơn, command này sẽ cài đặt qiskit-aer từ sdist (source distribution) và sdist sẽ dịch lại mã nguồn aer trên máy bạn.

Nếu các package đã được cài đặt đúng, bạn có thể chạy conda list để xem các package đang được kích hoạt trong môi trường ảo của bạn.

Nếu bạn định sử dụng thêm chức năng hiển thị đồ họa hoặc Jupyter Notebooks, bạn cần cần cài đặt qiskit với các tính năng mở rộng về hiển thị đồ họa

pip install qiskit[visualization]

Ngoài ra, nếu bạn sử dụng zsh (shell mặc định trên những phiên bản mới hơn của macOS), bạn cần đặt qiskit[visualization] trong dấu nháy đơn:

pip install 'qiskit[visualization]'

Ghi chú

Sau khi đã cài đặt và kiểm tra đúng các gói Qiskit bạn muốn sử dụng, import (khai báo sử dụng) chúng vào môi trường của bạn với Python để bắt đầu làm việc.

import qiskit

Ghi chú

Nếu bạn muốn đóng góp vào cộng đồng Qiskit bằng bằng việc phát triển và đưa code của mình vào Qiskit, tham khảo tại Build Qiskit packages from source.

Truy cập các hệ thống lượng tử của IBM¶

IBM Quantum offers several real quantum computers and high-performance classical computing simulators through its IBM Quantum Lab. Follow these steps to set up your Qiskit environment to send jobs to IBM Quantum systems.

Để cấu hình tài khoản, bạn cần tạo tập tin cấu hình cục bộ có chứa khoá API

1. Tạo tài khoản IBM Quantum miễn phí.

2. Sao chép token truy cập của bạn từ trang tổng quan của IBM Quantum.

3. Chạy các lệnh sau để lưu API token cục bộ trong tệp cấu hình được đặt tên qiskitrc. Thay thế MY_API_TOKEN bằng API token mà bạn đã copy ở trên.

from qiskit import IBMQ
IBMQ.save_account('MY_API_TOKEN')

Để biết thêm chi tiết, như cách quản lí nhiều tài khoản IBM Quantum, hãy tham khảo ở đây The IBM Quantum Account.

Kiểm tra phiên bản nào được cài đặt¶

Vì gói Qiskit chứa nhiều thành phần khác nhau, lệnh in qiskit.__version__ có thể gây hiểu nhầm khi nó chỉ trả về phiên bản cho gói qiskit-terra. Điều này là vì namespace qiskit trong Python không xuất phát từ gói Qiskit, mà thay vào đó là một phần của gói qiskit-terra.

import qiskit
qiskit.__version__

'0.16.4'

Để xem phiên bản của tất cả các thành phần của Qiskit trong môi trường cá nhâ, bạn có thể sử dụng thuộc tính __qiskit_version__. Ví dụ, chạy lệnh sau sẽ trả lại một từ điển bao gồm các phiên bản cho mỗi thành phần đã cài đặt của Qiskit.

qiskit.__qiskit_version__

{'qiskit-terra': '0.16.4',
 'qiskit-aer': '0.7.6',
 'qiskit-ignis': '0.5.2',
 'qiskit-ibmq-provider': '0.12.2',
 'qiskit-aqua': '0.8.2',
 'qiskit': '0.24.1'}

Mẹo

Nếu bạn đang phản hồi một vấn đề gặp phải, hoặc cần chia sẻ phiên bản Qiskit đã cài đặt cho một mục đích nào đó, sử dụng thuộc tính __qiskit_version__.
